
**Создание и настройка виртуальных машин с помощью VirtualBox** — это процесс, который позволяет запускать одну операционную систему внутри другой, а также настраивать ресурсы (процессор, память, видеоадаптер и др.). VirtualBox доступен для Linux, Solaris, macOS и Windows, скачать программу можно с официального сайта.
## Создание

**Чтобы создать виртуальную машину в VirtualBox, нужно**: 

1. Открыть программу и нажать «Создать» в правой части окна.
2. В появившемся окне ввести имя ОС и выбрать путь для хранения машины. Тип ОС будет определён автоматически на основе введённого названия.
3. Указать объём оперативной памяти для виртуальной машины.
4. Выбрать опцию «Создать новый виртуальный жёсткий диск» и нажать «Создать».
5. В настройках жёсткого диска выбрать формат VDI.
6. Установить тип диска: динамический (рекомендуется, так как его размер изменяется в зависимости от использования) или фиксированный.
7. Указать желаемый размер виртуального диска.

**Важно**: после создания виртуальной машины ещё не установлена операционная система — нужно выделить место для неё, но ещё не установить её.

## Настройка

**Некоторые параметры, которые можно настроить в VirtualBox:**

- **Процессор** — можно задать пользовательские настройки процессора, например, выбрать количество ЦП или установить предел загрузки ЦП.
- **Графика** — можно изменить объём видеопамяти (максимальное значение зависит от видеокарты, например, 256 МБ для карт с объёмом 2 ГБ).
- **Сеть** — по умолчанию виртуальная машина использует сеть NAT, что обеспечивает доступ к интернету. Для подключения нескольких виртуальных машин в одну сеть нужно сделать дополнительные настройки.
- **Клонирование** — позволяет создать резервную копию виртуальной машины, чтобы в случае проблем восстановить её. Для этого нужно щёлкнуть правой кнопкой мыши по виртуальной машине и выбрать «Клонировать…».
- **Запуск виртуальной машины** — для запуска понадобится загрузочный диск (образ ISO) операционной системы. Нужно выбрать созданную виртуальную машину и нажать «Запустить» в правой части окна, указать путь к образу операционной системы и установить ОС внутри VirtualBox.

### Виртуализация Android

**Создание и настройка виртуализации для Android** (виртуальных машин с Android) возможны с помощью программ-гипервизоров, например VirtualBox или Hyper-V. Эти инструменты позволяют запускать Android на компьютере, изолируя систему от реального устройства.

Ниже приведены примеры создания и настройки виртуальных машин с Android в VirtualBox и Hyper-V.

## В VirtualBox

**Создание виртуальной машины с Android**: 

- **Указать название**, тип и версию системы (например, Linux, Android x86).
- **Выделить объём оперативной памяти** — рекомендуется 1024 МБ.
- **Создать новый виртуальный жёсткий диск** — выбрать тип диска (VDI).
- **Указать вариант использования жёсткого диска**: можно зарезервировать всё место сразу или разрешить увеличивать размер диска по мере использования.

**Настройка виртуальной машины** (перед запуском): 
- **Настроить процессор**: установить 2 ядра, активировать PAE/NX.
- **Настроить дисплей**: установить видеопамять, включить 3D-ускорение.


**Установка Android**: 

- **Запустить виртуальную машину**.
- **В качестве загрузочного диска** указать образ с Android (например, скачанный в формате ISO).
- **Выбрать способ установки**: «Installation — Install Android-x86 to harddisk».
- **Создать раздел** для установки Android, назначить его главным (Primary).
- **Сделать раздел загрузочным** — задать ему параметр «Bootable».

## В Hyper-V

**Создание виртуальной машины с Android**: 

- **В «Диспетчере Hyper-V» создать новую виртуальную машину** — указать имя, поколение (например, Generation 1).
- **Установить объём оперативной памяти** для создаваемой машины.
- **Активировать опцию «Install an operating system from a bootable CD/DVD-ROM»**, указать путь к загруженному образу Android.
- **Завершить процесс создания виртуальной машины** — нажать «Finish».



**Установка Android**:

- **После создания виртуальной машины** выбрать её правой кнопкой мыши и нажать «Подключить».
- **В стартовом меню Android** выбрать «Installation — Install Android-x86 to harddisk».
- **Создать раздел** для установки Android, назначить его главным (Primary).
