

### Работа со струнами
Строки в Python — неизменяемые последовательности символов Unicode. Создаются в одинарных, двойных или тройных кавычках. Индексация с нуля, поддерживаются срезы и шаг. Изменять строки на месте нельзя: любые методы возвращают новые объекты. Для склейки множества частей используйте ''.join(список), а не оператор `+` в цикле. Основные методы: lower/upper, strip, split/join, replace, find/index, count, startswith/endswith, isdigit/isalpha. Форматирование: f-строки (рекомендуемый стандарт). При чтении/записи файлов и сетевом обмене всегда явно указывайте encoding='utf-8; преобразование в байты — `.encode()`, обратно — `.decode()`. Для сложного поиска и замены подключайте модуль `re`. Главные правила: f-строки для подстановки, `join` для конкатенации, явная кодировка, учёт неизменяемости строк.
