#  Множества (Sets) в Python

## Что это?

Множество (`set`) — это неупорядоченная коллекция уникальных(неповторяющихся), изменяемых элементов. Главная особенность — автоматическое удаление дубликатов.

## Создание множества

```python
# Пустое множество (внимание: {} создаст словарь!)
empty_set = set()

# С элементами
fruits = {'apple', 'banana', 'cherry'}
numbers = set([1, 2, 3, 2, 1])  # {1, 2, 3}

# Из строки
chars = set("hello")  # {'h', 'e', 'l', 'o'}
```

## Основные операции

### Добавление и удаление

| Операция | Код | Описание |
|----------|-----|----------|
| Добавить элемент | `s.add(x)` | Добавляет x, если его нет |
| Удалить элемент | `s.remove(x)` | Ошибка, если x не найден |
| Безопасное удаление | `s.discard(x)` | Не вызывает ошибку |
| Удалить случайный | `s.pop()` | Возвращает и удаляет элемент |
| Очистить всё | `s.clear()` | Удаляет все элементы |

### Проверка наличия (очень быстрая, O(1))

```python
if 'apple' in fruits:
    print("Есть!")
```

## Операции над множествами (как в математике)

```python
A = {1, 2, 3, 4}
B = {3, 4, 5, 6}
```

| Операция | Синтаксис | Метод | Результат |
|----------|-----------|-------|-----------|
| Объединение | `A \| B` | `A.union(B)` | {1,2,3,4,5,6} |
| Пересечение | `A & B` | `A.intersection(B)` | {3,4} |
| Разность | `A - B` | `A.difference(B)` | {1,2} |
| Сим. разность | `A ^ B` | `A.symmetric_difference(B)` | {1,2,5,6} |

### Проверка подмножеств

```python
{1, 2}.issubset({1, 2, 3})   # True
{1, 2, 3}.issuperset({1, 2}) # True
A.isdisjoint(B)               # False (если нет общих элементов)
```

## Преимущества множеств

1. Уникальность — автоматическое удаление дубликатов
   ```python
   unique_list = list(set([1, 2, 2, 3, 3, 3]))  # [1, 2, 3]
   ```

2. Скорость — проверка `in` работает за O(1) (в списке за O(n))

3. Математические операции — удобно для работы с пересечениями и т.д.

## Ограничения

- Элементы должны быть хешируемыми (неизменяемыми): числа, строки, кортежи. Нельзя добавить список, словарь или другое множество.
- Нет порядка элементов (до Python 3.7 тоже не было порядка, сейчас он сохраняет порядок вставки, но полагаться на это не стоит)

## FrozenSet — неизменяемое множество

```python
fs = frozenset([1, 2, 3])
# fs.add(4)  # ❌ Ошибка — нельзя изменить
```

Может использоваться как ключ словаря или элемент другого множества.

##  Реальные примеры использования

### 1. Удаление дубликатов из списка
```python
names = ['alex', 'john', 'alex', 'maria']
unique_names = list(set(names))
```

### 2. Поиск общих элементов в двух списках
```python
common = set(list1) & set(list2)
```

### 3. Проверка уникальности символов в строке
```python
def is_unique(s):
    return len(s) == len(set(s))
```

### 4. Группировка по уникальным значениям
```python
active_users = {user.id for user in users if user.is_active}
```

## Шпаргалка

```python
# Создание
s = set([1,2,3])      # из списка
s = {1,2,3}           # литерал

# Добавление/удаление
s.add(4)              # добавление
s.update([5,6,7])     # добавить несколько
s.remove(4)           # удалить (ошибка если нет)
s.discard(4)          # удалить (без ошибки)

# Операции
a | b                 # объединение
a & b                 # пересечение
a - b                 # разность
a ^ b                 # симметрическая разность
```
