### Введения

Класс - это пользовательский тип данных, конструкция ООП (Объектно Ориентированного Программирования), которая описывает тот или иной объект его свойства и способы взаимодействия с ним. 

Класс имеет поля переменные которые будут принадлежать к его конкретному экземпляру, а также методы - функции которые также будут принадлежать его экземпляру.

Простыми словами, класс - шаблон для реализации кого-то объекта. Например мы знаем как должна выглядеть кружка, какие характеристики и действия она имеет. При помощи класса мы описываем "абстрактную маму - кружку", а экземпляры будут являться настоящими или конкретными кружками, с характеристиками и действиями "мамы кружки".

Класс решает несколько важных задач

- Дублирование кода
- Унификация объектов(т.е. одинаковые по задумке предметы действительно будут иметь одинаковые конструкции и функционал)

### Классы Python

Объявляются с помощью ключевого слова class и указания в имени класса. Затем ставится двоеточие и со следующей строки описывается тело класса.

```
class Python:
	pass
````

Чтобы объявить поля класса используются ключевое слово self, после которого, через точку указывается имя поля. Поле тоже самое что и переменная, но внутри класса. Однако, через self можно обращаться только из тела метода. В параметрах метода тоже должен быть добавлен self. 

```
class Python:
	def test(self):
		self.msg = "Hello!"
```


Для инициализации полей при созданий класса используется функция - конструктор. Она также обязательно принимает параметр self. Эта функция будет автоматически вызвана при создании экземпляра класса. Если она не объявлена, будет использован конструктор по умолчанию(пустой).

```
class Python:
	#Конструктор !
	def __init__ (self):
		self.msg = "Hello"
```

Создание экземпляра класса(объект класса, объект) происходит вне тела класса, при помощи указания имени класса и круглых скобок после имени. Если в конструкторе кроме self указаны другие параметры, то при создании объекта мы так же укажем атрибуты - то, что является параметрами при описании функции.

```
Python:

class Python:
	#Конструктор !
	def __init__ (self):
		self.msg = "Hello"
		
python_exemple = Python("Hello")

```
