### База

- Строка — **неизменяемая** последовательность Unicode-символов
- Любая операция создаёт новую строку, оригинал не трогается
- Пустая строка `""` в условиях — `False`

```python
text = "кофе ☕️"
len(text)   # 6
```

### Индексация и срезы `[start:stop:step]`

```python
word = "python"
word[0]     # 'p'
word[-1]    # 'n'
word[0:3]   # 'pyt'
word[:3]    # 'pyt'
word[3:]    # 'hon'
word[::-1]  # 'nohtyp' (переворот)
word[::2]   # 'pto' (каждый второй)
```

Левая граница включена, правая — нет.

---

### Конкатенация и повторение

```python
"кот" + " " + "пёс"    # 'кот пёс'
"=" * 20               # '===================='
```

Не склеивать через `+` в цикле — лучше `"".join(list)`.

### Основные методы

### Регистр
```python
s.upper()        # ВЕРХНИЙ
s.lower()        # нижний
s.title()        # Каждое Слово С Заглавной
s.capitalize()   # Первое слово с заглавной
s.swapcase()     # Инвертировать рЕгИсТр
```

## Пробелы и символы
```python
s.strip()        # убрать пробелы с краёв
s.strip(",. ")   # убрать указанные символы
s.lstrip()       # слева
s.rstrip()       # справа
```

## Поиск
```python
s.find("что")      # индекс или -1
s.rfind("что")     # с конца
s.index("что")     # индекс или ValueError
s.count("что")     # количество вхождений
s.startswith("на") # True/False
s.endswith("ло")   # True/False
```

## Замена
```python
s.replace("старое", "новое")
s.replace("старое", "новое", 1)   # только первое вхождение
```

## Проверки (True/False)
```python
s.isalpha()    # только буквы
s.isdigit()    # только цифры
s.isalnum()    # буквы + цифры
s.isspace()    # только пробельные символы
s.islower()    # всё в нижнем
s.isupper()    # всё в верхнем
```

## Разбиение и склеивание
```python
"a,b,c".split(",")         # ['a', 'b', 'c']
"a,b,c".split(",", 1)      # ['a', 'b,c']
"a b  c".split()           # ['a', 'b', 'c'] (любые пробелы схлопываются)
"строка1\nстрока2".splitlines()   # по переносам строк

", ".join(["a", "b"])      # 'a, b'
```

### Форматирование

## f-строки (главный способ)
```python
name, age, money = "Анна", 25, 15200.5

f"Имя: {name}, возраст: {age}"
f"Баланс: {money:,.2f} руб."       # 'Баланс: 15,200.50 руб.'
f"{name:<10} | {age:>5}"          # выравнивание: '<' влево, '>' вправо, '^' центр
f"{name.upper()} авторизовалась"  # можно вызывать методы
```

## format (запасной вариант)
```python
"{} — {}".format("Анна", 25)
"{name} — {age}".format(name="Анна", age=25)
"{0} и ещё раз {0}".format("да")   # 'да и ещё раз да'
```

## Старый стиль (не использовать)
```python
"%s — %d" % ("Анна", 25)
```

### Спецсимволы

| Символ | Значение |
|--------|----------|
| `\n`   | перенос строки |
| `\t`   | табуляция |
| `\\`   | обратный слеш |
| `\'` `\"` | кавычки внутри |
| `r"..."` | сырая строка — экранирование не работает |

### Кодировки

```python
"Привет".encode("utf-8")          # строка → байты
b'\xd0\x9f...'.decode("utf-8")    # байты → строка
```

Всегда указывать кодировку при работе с файлами:
```python
open("file.txt", "r", encoding="utf-8")
```

### Типичные ошибки

```python
# 1 — забыл присвоить результат
text.strip()         # не работает, надо text = text.strip()

# 2 — изменить символ по индексу
s[0] = "x"           # TypeError, нельзя

# 3 — склеивание в цикле через +
result = ""
for x in data:
    result += x      # плохо, лучше через join
```
