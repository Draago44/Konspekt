## 🛠 Создание множеств

```python
# 1. С помощью фигурных скобок (элементы через запятую)
my_set = {1, 2, 3, 3, 4}  # Дубликаты автоматически удаляются -> {1, 2, 3, 4}

# 2. С помощью функции set() (например, из списка)
my_list = [1, 2, 2, 3]
my_set = set(my_list)     # -> {1, 2, 3}

# 3. Пустое множество (ВАЖНО: {} создает пустой словарь!)
empty_set = set()         
# type(empty_set) -> <class 'set'>
```

## ⚙️ Основные методы

| Метод | Описание | Пример |
|-------|----------|--------|
| `add(elem)` | Добавляет элемент в множество | `s.add(5)` |
| `remove(elem)` | Удаляет элемент. **Вызывает `KeyError`**, если элемента нет | `s.remove(5)` |
| `discard(elem)` | Удаляет элемент. **Не вызывает ошибку**, если элемента нет | `s.discard(5)` |
| `pop()` | Удаляет и возвращает **случайный** элемент. Вызывает `KeyError`, если множество пусто | `elem = s.pop()` |
| `clear()` | Очищает множество (делает его пустым) | `s.clear()` |
| `copy()` | Возвращает поверхностную копию множества | `new_s = s.copy()` |

## 🧮 Математические операции

Пусть `A = {1, 2, 3}` и `B = {3, 4, 5}`

### 1. Объединение (Union)
Все уникальные элементы из обоих множеств.
```python
A | B          # Оператор: {1, 2, 3, 4, 5}
A.union(B)     # Метод: {1, 2, 3, 4, 5} (принимает любой итерируемый объект)
```

### 2. Пересечение (Intersection)
Только те элементы, которые есть и в `A`, и в `B`.
```python
A & B                # Оператор: {3}
A.intersection(B)    # Метод: {3}
```

### 3. Разность (Difference)
Элементы, которые есть в `A`, но **нет** в `B`.
```python
A - B                # Оператор: {1, 2}
A.difference(B)      # Метод: {1, 2}
```

### 4. Симметрическая разность (Symmetric Difference)
Элементы, которые есть либо в `A`, либо в `B`, но **не в обоих сразу**.
```python
A ^ B                         # Оператор: {1, 2, 4, 5}
A.symmetric_difference(B)     # Метод: {1, 2, 4, 5}
```> 💡 **Совет:** Операторы (`|`, `&`, `-`, `^`) работают **только** с другими множествами. Методы (`.union()`, `.intersection()` и т.д.) принимают любые итерируемые объекты (списки, строки, кортежи).



##  Сравнение множеств

```python
A = {1, 2}
B = {1, 2, 3}
C = {1, 2}

# Равенство
A == C  # True

# Подмножество (A содержится в B)
A <= B          # True (или A.issubset(B))
A < B           # True (строгое подмножество, A != B)

# Надмножество (B содержит A)
B >= A          # True (или B.issuperset(A))
B > A           # True (строгое надмножество)

# Пересечение (есть ли хотя бы один общий элемент?)
A.isdisjoint(C) # False (у них есть общие элементы)
```

## 🔄 Генераторы множеств (Set Comprehensions)

Аналогично генераторам списков, но используют фигурные скобки.

```python
# Квадраты четных чисел от 0 до 9
squares = {x**2 for x in range(10) if x % 2 == 0}
# Результат: {0, 4, 16, 36, 64}
```

## 🧊 Неизменяемые множества (Frozenset)

`frozenset` — это неизменяемая (immutable) версия множества. 
**Зачем нужно:** Только хешируемые объекты могут быть ключами словаря или элементами другого множества. Обычный `set` нехешируем, а `frozenset` — хешируем.

```python
frozen = frozenset([1, 2, 3])

# frozen.add(4)  # ❌ AttributeError: 'frozenset' object has no attribute 'add'

# Использование как ключ словаря
my_dict = {frozen: "Это значение привязано к frozenset"}
```

## 🚀 Производительность и лучшие практики

1. **Сложность операций:** В среднем все основные операции (`add`, `remove`, `in`) выполняются за **O(1)** (константное время), так как под капотом используется хеш-таблица.
2. **Удаление дубликатов:** Самый быстрый и Pythonic-способ удалить дубликаты из списка:
   ```python
   unique_list = list(set(my_list))
   # Внимание: исходный порядок элементов не сохраняется!
   # Для сохранения порядка в Python 3.7+ используйте: list(dict.fromkeys(my_list))
   ```
3. **Проверка принадлежности:** Проверка `x in my_set` работает мгновенно (O(1)), в отличие от `x in my_list` (O(N)). Всегда конвертируйте большие списки в множества для частых проверок `in`.
4. **Элементы множества:** Могут быть только неизменяемыми типами: `int`, `float`, `str`, `tuple`, `frozenset`. Нельзя добавить `list`, `dict` или другой `set`.
