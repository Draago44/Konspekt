# Словари в Python

## Что это такое
Словарь это встроенная структура данных, которая хранит информацию в виде пар ключ и значение. Ключ работает как уникальный адрес или ярлык, по которому можно мгновенно найти нужное значение.

## Главные особенности
1. Ключи всегда уникальны. Если добавить новое значение по уже существующему ключу, старое значение просто перезапишется.
2. Ключом может быть только неизменяемый тип данных: строка, число или кортеж. Списки или другие словари ключами быть не могут.
3. Значением может быть любой тип данных: число, строка, список или даже другой словарь.
4. Начиная с Python 3.7, словарь автоматически запоминает и сохраняет тот порядок, в котором вы добавляли в него элементы.

## Как создать словарь
Пустой словарь создается с помощью пустых фигурных скобок или встроенной функции. Словарь с данными создается через фигурные скобки, где ключ и значение разделяются двоеточием, а разные пары разделяются запятыми.

## Основные операции
1. Получение значения по ключу происходит путем обращения к словарю с указанием нужного ключа.
2. Безопасное получение значения использует специальный метод. Он не вызовет ошибку, если ключа не существует, а вернет пустое значение или то, что вы укажете по умолчанию.
3. Добавление нового ключа или изменение существующего делается простым присваиванием значения нужному ключу.
4. Удаление элемента можно сделать командой удаления, которая полностью убирает ключ и его значение, или специальным методом, который удаляет ключ и одновременно возвращает его значение.

## Полезные методы
1. Метод keys возвращает все ключи словаря.
2. Метод values возвращает все значения словаря.
3. Метод items возвращает все пары ключ и значение.
4. Метод clear удаляет все элементы из словаря, оставляя его пустым.
5. Метод update объединяет текущий словарь с другим, обновляя существующие ключи и добавляя новые.

## Перебор словаря в цикле
Самый удобный и читаемый способ пройти по всем элементам это использовать цикл с методом items, который на каждом шаге выдает сразу и ключ, и значение. Если нужны только ключи или только значения, можно использовать методы keys или values соответственно.

## Проверка наличия ключа
Чтобы быстро узнать, есть ли определенный ключ в словаре, используйте специальный оператор. Это работает очень быстро, так как словари оптимизированы для мгновенного поиска по ключам.
