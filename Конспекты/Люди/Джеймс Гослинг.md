


### Основная информация
Полное имя: Джеймс Артур Гослинг (James Arthur Gosling)
Дата рождения: 19 мая 1955 года
Место рождения: около Калгари, Альберта, Канада
Известен как: «Отец Java»

### Образование
- Бакалавр наук в области компьютерных наук, Университет Калгари (1977)
- Доктор философии (PhD) в области компьютерных наук, Университет Карнеги-Меллона (1983)
- Диссертация: алгебраическое обобщение системы ограничений для редактора текстов

### Ключевые достижения
- Создание языка программирования Java (изначально носил название Oak) в 1995 году.
- Разработка оригинальной виртуальной машины Java (JVM) и компилятора.
- Создание текстового редактора Gosling Emacs (на языке C, для Unix).
- Участие в разработке оконной системы NeWS.
- Один из ключевых разработчиков IDE NetBeans.
- Автор спецификации для языка Java (JLS) и множества патентов в области вычислительной техники.

### Карьерный путь
- 1984–2010: Sun Microsystems. Начал с разработки оконных систем, позже возглавил проект Java.
- 2010–2011: Oracle Corporation. Кратковременная работа после поглощения Sun Microsystems.
- 2011–2012: Google. Занимался вопросами, связанными с разработкой и патентами.
- 2012–н.в.: Amazon Web Services (AWS). Работает в должности выдающегося инженера (Distinguished Engineer), фокусируясь на разработках для разработчиков и машинном обучении. Ранее также занимал пост технического директора в Liquid Robotics (компания была приобретена Boeing).

### Заметки и цитаты
«Лучше сделать что-то простое и заставить это работать, чем сделать что-то сложное и сломать это». 
Гослинг известен своим прагматичным подходом к разработке программного обеспечения, делая упор на надежность, переносимость и простоту использования.
