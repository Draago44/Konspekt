### .bat
Батник(от слова _batch-файл_) — это простой текстовый файл с расширением .bat или .cmd, который используется в операционной системе Microsoft Windows для автоматического выполнения команд.

# Кратко:

- Батник содержит набор команд командной строки.
    
- Эти команды выполняются по очереди через программу Command Prompt (cmd).
    
- Используется для автоматизации задач: запуск программ, копирование файлов, очистка папок и т.д.
    

# Пример батника:
~~~ python
@echo off
echo Привет!
pause
~~~
Этот файл:

1. отключает вывод команд,
    
2. выводит текст «Привет!»
    
3. ждёт нажатия клавиши.

### WinGet
WinGet — официальный пакетный менеджер от Microsoft для Windows 10/11, позволяющий управлять приложениями через командную строку.
# Источники пакетов

1. winget — открытый репозиторий сообщества на GitHub
2. msstore — Microsoft Store (приложения с рейтингом "E for Everyone")
3. Локальные манифесты — установка из локальных `.yaml`-файлов
4. Сторонние источники — можно добавить свои приватные репозитории

~~~ python
 # Установка для всех пользователей (требует прав администратора)
winget install -g <пакет>
# или
winget install --machine <пакет>

# Тихая установка (без интерфейса)
winget install <пакет> --silent

# Указание конкретной версии
winget install <пакет> --version 2.4.1

# Экспорт списка установленных программ
winget export -o packages.json

# Импорт и массовая установка из файла
winget import -i packages.json
~~~
##Требования и установка

 **Требования:**

- Windows 10 версии 1809 (сборка 17763) или новее
- 64-битная система
- Доступ к Microsoft Store (для автоматических обновлений)

 **Установка:**

- По умолчанию предустановлен в Windows 10 (2004+) и Windows 11
- Вручную: скачать с [GitHub Releases](https://github.com/microsoft/winget-cli/releases) или через:


```python
winget install Microsoft.**WinGet**
```
# Плюсы и минусы

 Преимущества:

- Официальный инструмент от Microsoft с интеграцией в ОС
- Открытый исходный код (MIT License)
- Поддержка автоматических обновлений
- Возможность скриптовой автоматизации
- Безопасность: проверка подписей и certificate pinning

 Ограничения:

- Не все приложения доступны в репозитории (каталог растёт, но меньше, чем у Chocolatey)
- Только 64-битные системы и Windows 10/11
- CLI-интерфейс может быть неудобен новичкам
- Некоторые функции всё ещё в разработке

```python
# Установить браузер
winget install Google.Chrome

# Обновить всё
winget upgrade --all

# Найти Python
winget search python

# Удалить программу
winget uninstall "OldApp"

# Показать, что можно обновить
winget **upgrade**
```
