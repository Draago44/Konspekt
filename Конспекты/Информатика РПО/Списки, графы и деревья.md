### Введение

Структура данных это программная единица, позволяющая хранить однотипные или логические связанные данные. Для добавления поиска изменения и удаления данных, для взаимодействия структура данных предоставляет некоторый набор функций представляющих её интерфейс.
### Связный список
Это базовая динамическая СД состоящая из узлов каждый из которых содержит значение и ссылку на следующий узел.
Двойной связанный список это список котором узлы содержат ссылки ещё и на предыдущий элемент.

### Деревья

Деревья это структура данных расширяющая идею связанного списка. Однако в отличии от них узлы могут иметь более одного последующего связанного с ними узла. Узлы дерева могут иметь любое количество дочерних узлов, благодаря чему данные представлены в виде гибкой схемы ветвления. 

### Графы

При помощи графов можно расширить область применения " Деревьев " и ослабить строгую родительскую часть в них. Узлы графа не имеют явной иерархии, каждый из узлов может быть соединен с любым другим узлом.
