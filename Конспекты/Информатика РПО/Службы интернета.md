### Определение 

Службы интернета - это протоколы и системы, которые обеспечивают обмен данными между устройствами. Современные службы работают по модели TCP / IP (пакетная передача данных).

...Ключевые принципы 

	1. Клиент - серверная модель, где клиент (пользыватель) делает запрос, а                         сервер на него отвечает .
	2. Порты - каждая служба использует собственный порт для обмена запросами.
	3. Безопасность - почти у каждой службы есть защищённая версия.

### Основные службы

1. DNS(англ. Domain Name System, рус. Служба Доменных Имён) - переводит удобные адреса сайтов в IP - адреса.
2. HTTP/HTTPS ( англ. HyperText Transfer Protocol, рус. Гипертекстовый Протокол Передачи) - основная служба для работы веб - сайтов. Через неё осуществляется передача данных: запросов от клиента и ответов от сервера. HTTPS - защищённая версия с шифрованием.
3. SMTP (англ. Simple Mail Transfer Protacol, Простой Протокол Передачи Почты) - протокол для отправки email. Является "почтальоном", который доставляет письма от сервера к получателю.


### Коммуникационные

- Электронная почта (E-mail) — служба передачи и получения электронных сообщений между пользователями. Протоколы: SMTP (отправка), POP3 и IMAP (получение).
- Мгновенные сообщения и чаты (IM, Instant Messaging) — обеспечивают обмен текстовыми сообщениями, файлами, иногда — голосовой и видеосвязью в режиме реального времени. Протоколы: IRC, XMPP, собственные протоколы мессенджеров (Telegram, WhatsApp, Viber и др.).
- Группы новостей и форумы (Usenet, Web-форумы) — службы для обсуждения тем в режиме асинхронной переписки, коллективной публикации сообщений по интересам. Протоколы: NNTP (для Usenet), веб-интерфейс (для форумов).



###### Информационные

- World Wide Web (WWW) — система связанных между собой гипертекстовых документов, к которым пользователь получает доступ через браузер. Веб-страницы объединяют текст, изображения, видео и звук, гипертекстовые ссылки связывают документы в единое информационное пространство. 
- FTP — специализированная служба для загрузки и скачивания файлов между клиентом и сервером. Протокол: FTP, SFTP (безопасный FTP). 
- Поисковые системы — позволяют находить информацию в интернете по определённым ключевым словам или запросам. Сканируют миллионы веб-страниц и предоставляют результаты поиска, основанные на запросе пользователя.
