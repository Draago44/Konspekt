
###  Пузырьковая сортировка (Bubble Sort)
Принцип: Попарное сравнение соседних элементов. Самые большие элементы «всплывают» в конец массива как пузырьки.  
Сложность: O(n²) — медленно, но просто для понимания.

~~~ python

def bubble_sort(arr):
	"""
	Сортировка пузырьком.
    На каждом проходе самый большой элемент встает на свое место в конце.
    """
    n = len(arr)
    # Внешний цикл: количество проходов (n-1 раз)
    for i in range(n - 1):
        # Флаг: были ли обмены на этом проходе (оптимизация)
        swapped = False
        print(f"\n--- Проход {i+1} ---")
        # Внутренний цикл: сравниваем пары до еще не отсортированной части
        # (n-i-1, потому что последние i элементов уже на месте)
        for j in range(n - i - 1):
            print(f"  Сравниваем {arr[j]} и {arr[j+1]}", end=" -> ")
            # Если левый больше правого -> меняем
            if arr[j] > arr[j+1]:
                arr[j], arr[j+1] = arr[j+1], arr[j]
                swapped = True
                print(f"Меняем! Массив: {arr}")
            else:
                print("Оставляем")
        # Если обменов не было — массив уже отсортирован, выходим
        if not swapped:
            print("  Обменов не было. Сортировка завершена.")
            break
    return arr
# Пример
test = [64, 34, 25, 12, 22, 11, 90]
print("Исходный массив:", test)
sorted_arr = bubble_sort(test.copy())
print("\nРезультат:", sorted_arr)
~~~
#### Как это работает:  
Алгоритм многократно проходит по массиву. На каждом проходе он сравнивает пары соседних элементов. Если левый элемент больше правого — они меняются местами. Представьте, что вы выстраиваете шеренгу людей по росту: вы проходите вдоль шеренги и каждый раз, когда видите, что высокий стоит перед низким, вы их меняете местами. После первого такого прохода самый высокий человек гарантированно окажется в конце.
### Сортировка выбором (Selection Sort)

Принцип: Делим массив на отсортированную и неотсортированную части. На каждом шаге ищем минимальный элемент в неотсортированной части и ставим его в начало этой части.  
Сложность: O(n²), но делает меньше обменов, чем пузырек.

~~~ python

def selection_sort(arr):
    """
    Сортировка выбором.
    Ищем минимум и ставим его на текущую позицию i.
    """
    n = len(arr)
    for i in range(n):
        # Предполагаем, что минимальный элемент стоит на позиции i
        min_index = i
        print(f"\n--- Ищем минимум для позиции {i} ---")
        # Ищем реальный минимум в оставшейся части массива
        for j in range(i + 1, n):
            if arr[j] < arr[min_index]:
                min_index = j
        # Меняем местами текущий элемент с найденным минимумом
        if min_index != i:
            print(f"  Меняем {arr[i]} (позиция {i}) и {arr[min_index]} (позиция {min_index})")
            arr[i], arr[min_index] = arr[min_index], arr[i]
        else:
            print(f"  Элемент {arr[i]} уже на своем месте (минимальный).")
        print(f"  Массив после шага {i+1}: {arr}")
    return arr
# Пример
test = [64, 25, 12, 22, 11]
print("Исходный массив:", test)
sorted_arr = selection_sort(test.copy())
print("\nРезультат:", sorted_arr)
~~~
#### Как это работает:  
Алгоритм делит массив на две части: отсортированную (слева) и неотсортированную (справа). Изначально отсортированная часть пуста. Алгоритм ищет минимальный элемент в неотсортированной части и меняет его местами с первым элементом неотсортированной части, тем самым расширяя отсортированную область.
### Сортировка вставками (Insertion Sort)

Принцип: Массив делится на отсортированную и неотсортированную части. Берём первый элемент из неотсортированной части и «вставляем» его в правильное место отсортированной части, сдвигая элементы вправо.  
Сложность: O(n²), но на почти отсортированных данных работает очень быстро (O(n)).

~~~ python

def insertion_sort(arr):
    """
    Сортировка вставками.
    Как при игре в карты: берем карту и вставляем в отсортированную стопку слева.
    """
    print("\n--- Начинаем сортировку вставками ---")
    # Начинаем со второго элемента (индекс 1), т.к. первый элемент считаем отсортированным
    for i in range(1, len(arr)):
        key = arr[i]  # Элемент, который нужно вставить
        j = i - 1     # Индекс последнего элемента в отсортированной части
        print(f"\nШаг {i}: Берем элемент {key} (индекс {i})")
        # Сдвигаем элементы отсортированной части вправо, пока они больше key
        while j >= 0 and arr[j] > key:
            arr[j + 1] = arr[j]
            print(f"  Сдвигаем {arr[j]} с позиции {j} на позицию {j+1}")
            j -= 1
        # Вставляем ключ на освободившееся место
        arr[j + 1] = key
        print(f"  Вставляем {key} на позицию {j+1}")
        print(f"  Массив: {arr}")
    return arr
# Пример
test = [12, 11, 13, 5, 6]
print("Исходный массив:", test)
sorted_arr = insertion_sort(test.copy())
print("\nРезультат:", sorted_arr)
~~~
#### Как это работает: 
Массив также делится на отсортированную и неотсортированную части. Алгоритм берет первый элемент из неотсортированной части (называемый "ключом") и находит ему место в отсортированной части, последовательно сдвигая все большие элементы вправо, освобождая место для вставки.
### Быстрая сортировка (Quick Sort) — Самая популярная

Принцип: Рекурсия. Выбираем опорный элемент (pivot). Делим массив на элементы меньше опорного и больше/равные опорному. Рекурсивно сортируем подмассивы.  
Сложность: O(n log n) в среднем, O(n²) в худшем (если неудачно выбрать опорный).

~~~ python

def quick_sort(arr):
    """
    Быстрая сортировка (разделяй и властвуй).
    Возвращает новый отсортированный массив.
    """
    # Базовый случай: массив из 0 или 1 элемента уже отсортирован
    if len(arr) <= 1:
        return arr
    # Выбираем опорный элемент (часто берут средний или последний)
    # Здесь возьмем последний для простоты
    pivot = arr[-1]
    left = []   # Элементы меньше опорного
    right = []  # Элементы больше опорного
    middle = [] # Элементы равные опорному (для устойчивости и скорости)
    print(f"\nРазделяем {arr} | Опорный: {pivot}")
    # Распределяем элементы по группам
    for x in arr:
        if x < pivot:
            left.append(x)
        elif x > pivot:
            right.append(x)
        else:
            middle.append(x)
    print(f"  left: {left}, middle: {middle}, right: {right}")
    # Рекурсивно сортируем левую и правую части и собираем результат
    return quick_sort(left) + middle + quick_sort(right)
# Пример
test = [33, 10, 55, 71, 29, 12, 10, 42]
print("Исходный массив:", test)
sorted_arr = quick_sort(test)
print("\nРезультат:", sorted_arr)
~~~
#### Как это работает:

1. Выбор опорного элемента (pivot). Обычно это первый, последний или средний элемент.
    
2. Разделение (partitioning). Массив перестраивается так, чтобы все элементы меньше опорного оказались слева от него, а все элементы больше — справа. Сам опорный элемент оказывается на своей финальной позиции.
    
3. Рекурсия. Рекурсивно применить те же шаги к левому и правому подмассивам.

### Сортировка слиянием (Merge Sort)

Принцип: Рекурсивно делим массив пополам, пока не останутся массивы из одного элемента. Затем сливаем их попарно в правильном порядке (как две упорядоченные колоды карт).  
Сложность: O(n log n) — стабильная и предсказуемая, но требует дополнительной памяти.

~~~ python

def merge_sort(arr):
    """
    Сортировка слиянием.
    Делим до 1 элемента, потом собираем склеивая отсортированные части.
    """
    if len(arr) <= 1:
        return arr
    # 1. Делим массив пополам
    mid = len(arr) // 2
    left_half = arr[:mid]
    right_half = arr[mid:]
    print(f"\nДелим: {arr} -> {left_half} | {right_half}")
    # 2. Рекурсивно сортируем половинки
    left_sorted = merge_sort(left_half)
    right_sorted = merge_sort(right_half)
    # 3. Сливаем две отсортированные половинки
    return merge(left_sorted, right_sorted)
def merge(left, right):
    """Функция слияния двух отсортированных массивов."""
    result = []
    i = j = 0
    print(f"  Сливаем {left} и {right}: ", end="")
    # Идем по двум массивам и берем меньший элемент
    while i < len(left) and j < len(right):
        if left[i] <= right[j]:
            result.append(left[i])
            i += 1
        else:
            result.append(right[j])
            j += 1
    # Добавляем оставшиеся элементы (если один массив кончился раньше)
    result.extend(left[i:])
    result.extend(right[j:])
    print(result)
    return result
# Пример
test = [38, 27, 43, 3, 9, 82, 10]
print("Исходный массив:", test)
sorted_arr = merge_sort(test)
print("\nРезультат:", sorted_arr)
~~~
#### Как это работает:

1. Разделение: Рекурсивно делим массив пополам, пока не получим подмассивы размером 1.
    
2. Слияние: Соединяем два соседних отсортированных подмассива в один отсортированный. Для этого создается временный массив, куда поочередно добавляются меньшие элементы из обоих подмассивов.

### Пирамидальная сортировка
Пирамидальная сортировка (Heapsort) — это эффективный алгоритм сортировки, основанный на структуре данных «куча» (heap). Алгоритм был предложен Дж. Уильямсом в 1964 году.

~~~ python

def heapify(arr, n, i):
    """
    Преобразует поддерево в max-heap
    
    Args:
        arr: массив
        n: размер кучи
        i: индекс корня поддерева
    """
    largest = i  # Изначально считаем корень наибольшим
    left = 2 * i + 1  # Левый потомок
    right = 2 * i + 2  # Правый потомок
    
    # Если левый потомок существует и больше корня
    if left < n and arr[left] > arr[largest]:
        largest = left
    
    # Если правый потомок существует и больше текущего наибольшего
    if right < n and arr[right] > arr[largest]:
        largest = right
    
    # Если наибольший элемент не корень
    if largest != i:
        arr[i], arr[largest] = arr[largest], arr[i]  # Меняем местами
        heapify(arr, n, largest)  # Рекурсивно преобразуем затронутое поддерево
def heap_sort(arr):
    """
    Пирамидальная сортировка
    
    Args:
        arr: список для сортировки
    
    Returns:
        отсортированный список (сортируется на месте)
    """
    n = len(arr)
    
    # Этап 1: Построение кучи (max-heap)
    # Начинаем с последнего родителя и идем к корню
    for i in range(n // 2 - 1, -1, -1):
        heapify(arr, n, i)
    
    # Этап 2: Извлечение элементов из кучи
    for i in range(n - 1, 0, -1):
        # Перемещаем текущий корень в конец
        arr[0], arr[i] = arr[i], arr[0]
        # Восстанавливаем кучу для уменьшенной кучи
        heapify(arr, i, 0)
    
    return arr
# Пример использования
if __name__ == "__main__":
    test_array = [12, 11, 13, 5, 6, 7]
    print("Исходный массив:", test_array)
    
    sorted_array = heap_sort(test_array.copy())
    print("Отсортированный массив:", sorted_array)
    
    # Дополнительный тест
    test_array2 = [4, 10, 3, 5, 1]
    print("\nИсходный массив:", test_array2)
    print("Отсортированный массив:", heap_sort(test_array2))
~~~
#### Как это работает?

Алгоритм состоит из двух основных этапов:

##### Этап 1: Построение кучи (heapify)

Превращаем исходный массив в двоичную кучу. Обычно используется **max-heap** (родитель больше детей), чтобы потом извлекать максимальные элементы.

##### Этап 2: Сортировка

1. Меняем местами корень кучи (максимальный элемент) с последним элементом массива
    
2. Уменьшаем размер кучи на 1
    
3. Восстанавливаем свойство кучи для корня (просеиваем вниз)
    
4. Повторяем, пока не останется один элемент
